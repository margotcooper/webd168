About This Page

This page was created using Bootstrap. When that was not sufficient, SCSS was used to customize the page even further. Bootstrap and SCSS are amazing for the following reasons:

  • Bootstrap allow for developers to implement design and function elements without requiring them to build such features from the ground up.
  • Bootstrap's ease of use makes it so that anyone with fundamental coding knowledge is able to build a sophisticated product.
  • Bootstrap's pre-programmed elements allow for optimization and reusability.
  • SCSS is arguably simpler and quicker to produce (than CSS), as well as makes stylesheet code easier to maintain.
